# Python Port Scanner

A simple port scanner built with Python to learn cybersecurity fundamentals.

## Features

- Scans ports 1-1024
- Shows open ports on any target
- Fast and lightweight

## Requirements 

- Python 3.x
- No external dependencies

## Usage

'''bash
python3 scanner.py
'''

'''
scanme.nmap.org
'''
